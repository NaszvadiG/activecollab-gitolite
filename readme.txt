Thank you for purchasing Ac Gitolite Module for activeCollab.


------------------------------------------------
Installation instructions:
------------------------------------------------

1. Unzip the Ac Gitolite Module zip file and upload ac_gitolite 
folder to your activeCollab server using FTP or SFTP client. Make sure you 
upload the folder in your activeCollab\custom\modules\ folder.

2. Make sure you upload all files within "ac_gitolite" folder.

3. Go to activeCollab Administration -> Modules. Install Ac Gitolite Module. 

4. That's all. You can now access Ac Gitolite from Administration.


NOTE: This module is tested on activecollab version 3.1.17.


Here's a demo link for module
http://ac.demo.rtcamp.com/


------------------------------------------------
Uninstallation / Removal instructions:
------------------------------------------------

- Uninstall Ac Gitolite module from activeCollab Administration -> Modules
